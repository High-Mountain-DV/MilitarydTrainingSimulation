## 💫 프로젝트 소개 ##
"군사 훈련 플랫폼 <p>
이 프로젝트는 메타버스 아카데미의 XR-언리얼, 백엔드(back-end), AI, TA 팀이 협업하여 진행한 프로젝트입니다. <br>


## 🚀 기획 내용 ##
‘높은 산 깊은 골’은 메타버스 환경에서 사용자가 직접 제작 가능한 VR 군사 훈련을 제공하는 서비스 플랫폼입니다. <p><br>
